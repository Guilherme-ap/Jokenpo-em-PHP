# Jogo de Jokenpô em PHP

![Jokenpô](jokenpo.png)

Um simples jogo de Jokenpô desenvolvido em PHP. Este é um projeto inicial que demonstra os conceitos básicos do PHP e como usar cookies para armazenar informações entre as requisições.

## Funcionalidades

- O jogo permite que o usuário escolha uma das opções: Pedra, Papel ou Tesoura.
- O computador faz uma escolha aleatória.
- As escolhas do usuário e do computador são comparadas para determinar o vencedor.
- O resultado do jogo é exibido na tela.
- O cookie é usado para armazenar o placar do usuário.

## Requisitos

- XAMPP (ou qualquer servidor web com suporte a PHP)
- PHP 7.0 ou superior
