<h1>Jogo de Jokenpô em PHP</h1>

<img src="https://i0.wp.com/heroisx.com/wp-content/uploads/2015/07/Jo-Ken-Po.jpg" alt="Jokenpô">

<p>Um simples jogo de Jokenpô desenvolvido em PHP. Este é um projeto inicial que demonstra os conceitos básicos do PHP e como usar cookies para armazenar informações entre as requisições.</p>

<h2>Funcionalidades</h2>

<ul>
  <li>O jogo permite que o usuário escolha uma das opções: Pedra, Papel ou Tesoura.</li>
  <li>O computador faz uma escolha aleatória.</li>
  <li>As escolhas do usuário e do computador são comparadas para determinar o vencedor.</li>
  <li>O resultado do jogo é exibido na tela.</li>
  <li>O cookie é usado para armazenar o placar do usuário.</li>
</ul>

<h2>Requisitos</h2>

<ul>
  <li>XAMPP (ou qualquer servidor web com suporte a PHP)</li>
  <li>PHP 7.0 ou superior</li>
</ul>

<h2>Instalação</h2>

<ol>
  <li>Baixe e instale o XAMPP seguindo as instruções fornecidas em: <a href="https://www.apachefriends.org/index.html">https://www.apachefriends.org/index.html</a></li>
  <li>Clone este repositório:</li>
</ol>

<pre><code>git clone https://github.com/Guilherme-ap/Jokenpo-em-PHP
</code></pre>

<ol start="3">
  <li>Mova o diretório do jogo para a pasta "htdocs" do seu XAMPP:</li>
</ol>

<pre><code>mv jogo-jokenpo-php /opt/lampp/htdocs/
</code></pre>

<ol start="4">
  <li>Inicie o servidor Apache e o servidor MySQL no painel de controle do XAMPP.</li>
  <li>Abra o jogo no seu navegador em <a href="http://localhost/jogo-jokenpo-php">http://localhost/jogo-jokenpo-php</a>.</li>
</ol>
